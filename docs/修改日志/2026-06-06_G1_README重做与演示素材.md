## 抉择
- **GIF 本身需人工录制**：环境无法屏录/交互录制，故交付「README + 占位图 + 一键 seed + 录制指南」
  这套**录制就绪**脚手架，并在图注/assets 指南里明确「占位、待替换」，不谎称已有 GIF。
- **占位图用 SVG 而非留空**：SVG 文本清晰、体积小、GitHub 直接渲染，避免裂图；替换成本低。
- **seed 自包含、不依赖 tests/sample_data**：演示工具与测试解耦，内联一份简历+2 JD，路径无关。
- **seed 只清三张状态表**：不动简历与历史分析，可反复运行得到干净的单条演示旅程。
- **G2 一页架构说明暂未单独成文**：架构亮点已浓缩进 README「架构概览」段；如需独立 G2 再拎出。
